【機能】
UV 平面上における任意の2点をそれぞれ拡大縮小中心、回転中心として、
UV 選択部を拡大縮小・回転・平行移動します。
Scaling, Rotation, Translation (SRT) の各パラメータはダイアログ上で入力します。
クリップボードを経由してデータをやり取りすることもできます。


【インストール】
Metasequoia をインストールしたフォルダーの下の
「Plugins/Select」フォルダーに、本プラグイン DLL ファイル（※）を配置します。
または、Metasequoia メニューの[ヘルプ]→[プラグインについて]からダイアログを開き、
[インストール]ボタンで本プラグイン DLL ファイル（※）を選びます。
32bit 版 Metasequoia の場合は Win32 フォルダー配下の DLL を、
64bit 版 Metasequoia の場合は x64 フォルダー配下の DLL を配置するようにしてください。


【使い方】
[選択部処理]→[UV Matrix]を実行します。
ダイアログが表示されるので、各パラメータを入力して、
[実行]ボタンを押してください。

また、「ホワイトスペースで区切られた9つの数値」を表すテキストがクリップボードに
存在するならば、「クリップボードから値を読み取る」ボタンを押すことで、
一発で座標を入力できます。
逆に「クリップボードに値を書き込む」ボタンを押すことで、現在入力されている
数値をクリップボードに書き込みます。
メモ帳などのテキストエディターで編集したテキストからデータ入力する場合、
あるいはデータをテキストに保存して加工・再利用する際に便利です。
データの並びは、
"拡縮中心U 拡縮中心V 拡縮率U 拡縮率V 回転中心U 回転中心V 回転量 平行移動量U 平行移動量V"
の順になっています。

　テキストの例：0.5 0.3 1.5 3.0 -0.2 -0.1 -45.0 0.3 0.4
　ホワイトスペースとは、半角スペース 0x20 やタブ文字 0x09、改行 0x0a などを指します。
　ホワイトスペースならばいくら入っていてもかまいません。
　また、1.0e3 (1.0×10の3乗) のような指数表現も可能です。
　クリップボードに書き込む際は、区切り文字として改行 0x0a が使われます。
　なお、本プラグインは Unicode 対応しているため、ホワイトスペースとして全角スペース U+3000H も使用可能ですが、
　数値部分には半角英数 (ASCII コード) しか使えません。

***** UV 選択とは？ *****
Metasequoia において、通常選択と UV 選択は異なることに注意してください。
ある頂点が通常選択されていても、UV 選択されているとは限りません。
その逆もまた然りです。
頂点が UV 選択されていると、Metasequoia の[UV操作]コマンドで展開された
UV 展開図上において、頂点が選択された状態で表示されます。

本プラグインは、通常選択ではなく UV 選択されている頂点に対してのみ機能します。
UV 選択された頂点がカレントオブジェクト内にひとつもない場合、
本プラグインを実行してもダイアログは表示されません。

***** 値の誤差について *****
Metasequoia およびプラグイン内部で使用されている32ビット浮動小数点数の精度の限界のため、
数値の有効桁数は6～7桁となります。
したがって、桁数が6桁を超えるデータをクリップボードから取得する際に
7桁目以降のデータが正確に取得できません。
これは仕様なのであらかじめご了承ください。


【多角形面について】
Metasequoia はバージョン 4 にて五角形以上の多角形面に対応しましたが、
旧バージョンの SDK を使って作成されたプラグインを多角形面に対して実行しようとすると、
「古いプラグインを実行するには多角形を分割する必要があります。
多角形を分割してプラグイン処理を実行しますか？」
というメッセージが表示され、多角形を維持したまま処理を適用することができません。
本プラグインはソースコードレベルでは多角形面に対応しており、
新しい SDK を使ってビルドすることで多角形面に対応することもできるはずですが、
もともと旧バージョンとの互換性を維持したうえでメンテナンスしているプラグインのため、
オフィシャルには多角形面には対応しません。
多角形面に対応したい場合は自前で SDK を差し替えてビルドするなりしてください。
