オブジェクトプラグイン。

ただし、OS サポートは日本語版 Windows 7 以降のみとなります。

【機能】
あるオブジェクトから別のオブジェクトに UV を移植します。
移植元・移植先オブジェクトは、トポロジーが全く同じである必要があります。
詳しくは位相幾何学のテキストに譲りますが、簡単に説明すると、
頂点数・面数が同じで、なおかつ各面のインデックスが移植元・移植先で同一
（面のつながり方、連結性が同一）であればよい、ということです。
絶対的な頂点位置座標および UV 座標（ジオメトリ）は異なっていてもかまいません。
また、オブジェクト内に複数のマテリアルが含まれていたり、移植元と移植先で
マテリアルが異なっていたりしてもかまいませんが、移植元・移植先ともに
UV マッピングが施されている必要があります。

※多足動物などのモデリングにおいて、UV マッピング座標を後から調整する場合に
この UV 移植機能を利用すると効果的かもしれません。


【インストール】
Metasequoia をインストールしたフォルダーの下の
「Plugins/Object」フォルダーに、本プラグイン DLL ファイル（※）を配置します。
または、Metasequoia メニューの[ヘルプ]→[プラグインについて]からダイアログを開き、
[インストール]ボタンで本プラグイン DLL ファイル（※）を選びます。
32bit 版 Metasequoia の場合は Win32 フォルダー配下の DLL を、
64bit 版 Metasequoia の場合は x64 フォルダー配下の DLL を配置するようにしてください。


【使い方】
[オブジェクト]→[UV Implant]を実行します。
ダイアログが表示されるので、移植元・移植先オブジェクトを選択後、
[実行]ボタンを押してください。
失敗すればエラーメッセージが表示されますが、成功すれば何も表示されません。


【多角形面について】
Metasequoia はバージョン 4 にて五角形以上の多角形面に対応しましたが、
旧バージョンの SDK を使って作成されたプラグインを多角形面に対して実行しようとすると、
「古いプラグインを実行するには多角形を分割する必要があります。
多角形を分割してプラグイン処理を実行しますか？」
というメッセージが表示され、多角形を維持したまま処理を適用することができません。
本プラグインはソースコードレベルでは多角形面に対応しており、
新しい SDK を使ってビルドすることで多角形面に対応することもできるはずですが、
もともと旧バージョンとの互換性を維持したうえでメンテナンスしているプラグインのため、
オフィシャルには多角形面には対応しません。
多角形面に対応したい場合は自前で SDK を差し替えてビルドするなりしてください。
